# 常见技术的spring-boot集成

## 目前已完成集成案例
* springboot + mybatis plus
    1. 支持 mybatis plus 的增删改查
    2. 支持 mybatis的分页的统一处理
    3. 支持 统一返回的 VO 实现
    4. 支持 自定义异常枚举类，及异常的全局统一处理
* springboot + clickhouse +  druid + mybatis plus
    1. 支持 mybatis plus 对 clickhouse 的增删改查操作
